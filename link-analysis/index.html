Link Analysis and PageRank

Exam-ready explanation of link analysis, the PageRank algorithm, modified PageRank, damping factor, and why teleportation matters.

Overview / 总览

Exam core idea: Link analysis treats documents, people, or pages as nodes in a graph. PageRank assigns each node an importance score by letting important pages pass rank through their outgoing links. Modified PageRank adds teleportation and sink handling so the score is stable, normalized, and not trapped.

中文：核心考点是：把网页、人、论文等对象看成图里的节点，把它们之间的连接看成边。PageRank 让重要节点通过出边传递分数；改进版 PageRank 加入随机跳转和 sink 处理，让分数稳定、总和保持为 1，并且不会卡住。

中文理解

把网页看成一个有向图：网页是点，超链接是箭头。一个网页被很多重要网页指向，就更重要。PageRank 不是只数入边数量，而是问："给你投票的网页本身有多重要？它把票分给了多少网页？"

English Exam Phrase

Link analysis evaluates relationships among nodes in a graph. PageRank is a query-independent authority score that ranks pages using the number and quality of incoming links, with each linking page splitting its importance across its outbound links.

中文：Link analysis 分析图中节点之间的关系。PageRank 是一种不依赖查询内容的权威性分数，它用入链的数量和质量给网页排名，并且每个链接来源会把自己的重要性分给它的出链。

What Is Link Analysis? / 什么是链接分析？

Definition / 定义

中文理解
Link Analysis 是一种分析网络关系的方法。它研究节点之间的连接，比如网页之间的超链接、社交网络里的关注关系、论文之间的引用关系，从而发现重要节点、群体结构、隐藏关系。

English answer
Link analysis is a data-analysis technique that evaluates relationships between nodes in a graph to identify patterns, measure importance, and uncover hidden connections.

中文：考试回答时可以说：Link Analysis 用图结构中的连接关系来找模式、衡量重要性、发现隐藏关系。

Why Links Matter / 为什么链接重要

A hyperlink is a quality signal. If page d1 links to page d2, the author of d1 is implicitly saying that d2 is relevant, useful, or authoritative.

中文：超链接可以看作质量信号。如果 d1 链接到 d2，就表示 d1 的作者认为 d2 有相关性、有用或有权威性。

Important caution: a raw count of incoming links can be manipulated, so PageRank also considers the importance of the pages that link to you.

中文：注意：只数入链数量很容易被操纵，所以 PageRank 不只看有多少网页链接你，还看链接你的网页本身有多重要。

What Is the PR Algorithm? / 什么是 PR 算法？

PR means PageRank. It assigns an importance score to every page. The score is query-independent: it measures static authority from the link graph, not whether the page matches a particular search query.

中文：PR 就是 PageRank。它给每个网页一个重要性分数。这个分数和用户具体搜什么无关，只衡量网页在链接图里的静态权威性。

Simplified PageRank update rule / 简化版 PageRank 更新规则

PR(u) = Σv ∈ In(u) PR(v) / Out(v)

A page u receives rank from every incoming page v. Each v divides its old rank equally among the pages it links to.

中文：页面 u 的新分数来自所有指向它的页面 v。每个 v 会把自己上一轮的分数平均分给它链接出去的页面。

  • PR(u): importance score of page u.中文：页面 u 的重要性分数。
  • In(u): set of pages that link to u.中文：所有指向 u 的页面集合。
  • Out(v): number of outgoing links from page v.中文：页面 v 指出去的链接数量。
  • N: total number of pages.中文：总页面数。

Algorithm Steps / 算法步骤

  1. Initialize each page with rank 1 / N, so the total rank over all pages is 1.中文：一开始每个页面分数都是 1 / N，保证所有页面分数加起来等于 1。
  2. For every page u, compute a new rank from the previous iteration: add up PR(v) / Out(v) for every page v that points to u.中文：对每个页面 u，把所有指向 u 的页面 v 的贡献加起来：每个贡献是 PR(v) 除以 v 的出链数量。
  3. Repeat the update until the ranks stop changing much. This is convergence.中文：不断重复更新，直到分数变化很小，这叫收敛。
  4. Sort pages by final PageRank. Higher PageRank means higher link-based authority.中文：最后按 PageRank 从高到低排序，分数越高表示链接结构里的权威性越高。

中文理解

普通 PageRank 就像不断传递"重要性的钱"。每个网页把自己的分数平均分给它链接出去的网页。收到越多来自重要网页的分数，你的 PageRank 越高。

English Exam Phrase / 英文考试句

PageRank recursively defines a page's importance as the sum of the importance contributed by its incoming neighbors, where each neighbor splits its rank among its outbound links.

中文：PageRank 递归地定义重要性：一个页面的重要性来自指向它的邻居页面，而每个邻居会把自己的分数分给所有出链。

What Is the Modified PR Algorithm? / 什么是改进版 PR 算法？

The modified algorithm fixes a major weakness of simplified PageRank: a random surfer can get trapped in disconnected components or lose probability mass at pages with no outgoing links. Modified PageRank adds teleportation.

中文：改进版 PageRank 修复普通 PageRank 的问题：随机浏览者可能被困在不连通的部分，或者遇到没有出链的页面导致分数流失。改进版加入 teleportation，也就是随机跳转。

Modified PageRank with teleportation / 带随机跳转的改进版 PageRank

PR(pi) = (1 - d) / N + d · Σpj ∈ In(pi) PR(pj) / |Out(pj)|

The first term gives every page a small base probability from teleportation. The second term gives rank from incoming links when the surfer follows a link.

中文：第一项 (1 - d) / N 是每个页面都会得到的随机跳转基础分。第二项是用户按链接走时，从入链页面传来的 PageRank。

  • p1, p2, ..., pn: all pages considered.中文：当前图中所有被考虑的页面。
  • In(pi): pages that link to pi.中文：所有指向 pi 的页面。
  • |Out(pj)|: number of outbound links from pj.中文：pj 的出链数量。
  • d: damping factor, the probability of following a link.中文：阻尼因子，表示继续点击链接的概率。
  • 1 - d: probability of teleporting uniformly to any page.中文：随机跳到任意页面的概率。
  • N: total number of pages, used to distribute teleport probability uniformly.中文：页面总数，用来平均分配随机跳转概率。

Modified Algorithm Steps / 改进版算法步骤

  1. Initialize every page to 1 / N.中文：每个页面初始分数设为 1 / N。
  2. Choose a damping factor, usually d = 0.85.中文：选择阻尼因子，常见取值是 d = 0.85。
  3. For each page, add the teleportation term (1 - d) / N.中文：每个页面先加上随机跳转带来的基础分。
  4. Add the link-following contribution from incoming neighbors, multiplied by d.中文：再加上入链页面传来的分数，并乘以 d。
  5. Repeat until convergence.中文：重复计算直到分数稳定。

What Is Damping Factor? / 什么是阻尼因子？

Definition / 定义

中文理解
Damping factor 记作 d，表示随机浏览者继续点击当前页面链接的概率。1 - d 表示他不再沿着链接走，而是随机跳到任意一个网页。

English answer
The damping factor is the probability that the random surfer follows an outgoing link instead of teleporting to a random page. In PageRank, it controls the balance between link structure and random exploration.

中文：一句话记：d 控制“相信链接结构”的程度；1 - d 控制“随机探索”的程度。

Typical Value / 常用取值

Most PageRank explanations use d = 0.85. That means a surfer follows a link 85% of the time and teleports 15% of the time.

中文：常用 d = 0.85，意思是 85% 的时候继续点链接，15% 的时候随机跳到其他页面。

Using the slide convention, expected steps until teleport are 1 / (1 - d). For d = 0.85, this is 1 / 0.15 = 6.67, matching the idea that users follow about 6-7 links before jumping away.

中文：按照课件里的解释，随机跳转前的期望步数是 1 / (1 - d)。当 d = 0.85 时，期望约为 6.67，接近用户平均点 6 到 7 个链接后离开的行为。

Geometric intuition for d = 0.85 / d = 0.85 的几何分布直觉

E[X] = 1 / p = 1 / (1 - d) = 1 / 0.15 ≈ 6.67

Here "success" means teleporting. Larger d means longer link-following chains. Smaller d means more random jumps and less dependence on link structure.

中文：这里的“成功”指发生随机跳转。d 越大，用户沿链接走得越久；d 越小，随机跳转越频繁，PageRank 越接近平均分。

Why Is Damping Factor Important? / 为什么阻尼因子重要？

Reason中文：原因 What it means中文：含义 Exam wording中文：考试表达

Sinks and the Final PageRank Formula / Sink 与最终 PageRank 公式

A sink, also called a dangling node, is a page with no outgoing links. If we do not handle it, rank can accumulate there and stop flowing through the graph.

中文：Sink 也叫 dangling node，指没有出链的页面。如果不处理，PageRank 可能流进这个页面后就不再流出，导致分数被吸住。

Dangling-aware PageRank / 处理 dangling node 的 PageRank

PR(pi) = (1 - d) / N + d ( Σpj ∈ In(pi) PR(pj) / |Out(pj)| + Σpk ∈ D PR(pk) / N )

D is the set of dangling nodes. Their rank is redistributed evenly to all pages, as if each sink had outgoing links to every page.

中文：D 是所有 dangling nodes 的集合。它们的分数会平均分给所有页面，效果等价于每个 sink 都连向所有页面。

Two Equivalent Fixes / 两种等价修复方法

Method 1: Add outgoing edges from each sink to every page, including itself, before the main loop.

中文：方法 1：在主循环开始前，给每个 sink 加上指向所有页面的出边，包括指向它自己。

Method 2: During each update, distribute each sink's PageRank evenly among all N pages.

中文：方法 2：每次更新时，把 sink 的 PageRank 平均分配给全部 N 个页面。

Common Mistake / 常见错误

Do not forget sink rank. If a dangling page has rank 0.25 and N = 4, each page receives an extra 0.25 / 4 inside the link-following part of the formula, then the whole link contribution is multiplied by d.

中文：常见错误是忘记 sink 的分数。比如一个 dangling page 的分数是 0.25，N = 4，那么每个页面会额外收到 0.25 / 4，这部分也属于括号里的链接贡献，最后还要乘 d。

Practice Q&A / 练习问答

1. What is link analysis? / 什么是链接分析？

Link analysis is a graph-based data-analysis technique that studies relationships between nodes. In web search, pages are nodes and hyperlinks are directed edges. The goal is to identify important pages, communities, or hidden relationships from the structure of the graph.

中文：Link analysis 是基于图的数据分析方法，研究节点之间的关系。在搜索里，网页是节点，超链接是有向边。目标是从图结构里找重要网页、群体或隐藏关系。

2. Why is in-degree alone not enough for ranking? / 为什么只看入度不够？

In-degree only counts how many pages link to a page. It ignores whether those pages are important and can be manipulated by creating many low-quality pages that all link to the target. PageRank improves this by weighting links from important pages more heavily.

中文：In-degree 只看有多少页面链接到你，不看这些页面重不重要，而且容易通过制造很多低质量页面来刷分。PageRank 会让重要页面的链接权重更高。

3. What does PageRank measure? / PageRank 衡量什么？

PageRank measures the importance or authority of a page based on the link graph. It is query-independent, meaning it does not depend on the user's specific search terms.

中文：PageRank 衡量网页在链接图中的重要性或权威性。它不依赖具体查询，也就是说用户搜什么不会改变这个基础权威分。

4. Explain the simplified PageRank formula in words. / 用文字解释简化版公式。

A page's new rank is the sum of rank contributions from every page linking to it. Each incoming page divides its previous rank equally across all pages it links to.

中文：一个页面的新分数等于所有指向它的页面贡献之和。每个入链页面会把自己上一轮的分数平均分给它所有出链。

5. What problem does modified PageRank solve? / 改进版 PageRank 解决什么问题？

It solves the problem that a random surfer may get trapped in disconnected components, cycles, or sinks. By adding teleportation, every page has a chance to receive rank even without incoming links.

中文：它解决随机浏览者被困在不连通部分、循环或 sink 里的问题。加入 teleportation 后，就算页面没有入链，也有机会获得分数。

6. What is damping factor? / 什么是阻尼因子？

The damping factor d is the probability that the random surfer follows a link from the current page. The probability 1 - d is the chance of teleporting to a random page. A typical value is d = 0.85.

中文：Damping factor d 表示随机浏览者继续点击当前页面链接的概率。1 - d 表示随机跳到任意页面的概率。常用 d = 0.85。

7. Why is damping factor important? / 为什么阻尼因子重要？

It prevents rank from getting trapped, helps the algorithm converge to stable scores, models realistic user behavior, and balances link-graph authority with random exploration.

中文：它可以防止分数卡住，帮助算法收敛，模拟真实用户不会一直点链接的行为，并平衡链接权威性和随机探索。

8. What is a sink or dangling node? / 什么是 sink/dangling node？

A sink is a page with no outgoing links. It can absorb PageRank because rank enters the page but cannot leave through links. PageRank handles this by redistributing sink rank uniformly to all pages.

中文：Sink/dangling node 是没有出链的页面。分数可以流进它，但不能通过链接流出，所以需要把它的分数平均分给所有页面。

9. What is the difference between relevance score and importance score? / 相关性分数和重要性分数有什么区别？

Relevance measures how well a document matches a query. Importance measures how authoritative the document is independent of the query. PageRank is an importance score, not a relevance score.

中文：Relevance 看文档和查询是否匹配；Importance 看文档本身是否权威、可信。PageRank 是 importance score，不是 relevance score。

10. Give a short exam answer for PageRank. / 写一个简短考试答案。

PageRank is a link-analysis algorithm that ranks pages by interpreting links as endorsements. A page is important if many important pages link to it. Each linking page distributes its rank across its outgoing links, and the modified version adds teleportation using damping factor d to ensure stable rankings.

中文：PageRank 是一种链接分析算法，把链接看作推荐/投票。如果很多重要页面链接到某页面，该页面就重要。每个页面会把自己的分数分给出链；改进版用 damping factor d 加入随机跳转，让排名更稳定。

Common Mistakes / 常见错误

Do not say PageRank only counts links. It counts number and quality of links.

中文：不要说 PageRank 只是数链接。它同时看链接数量和链接质量。

Do not confuse relevance with authority. PageRank is query-independent authority, not query matching.

中文：不要把相关性和权威性混淆。PageRank 是不依赖查询的权威性，不是查询匹配。

Do not forget division by outgoing links. A high-rank page with many outgoing links gives each target a smaller share.

中文：不要忘记除以出链数量。一个高分网页如果出链很多，每个目标拿到的份额会变小。

Do not forget teleportation. The modified formula always includes (1 - d) / N.

中文：不要忘记随机跳转。改进版公式总是包含 (1 - d) / N。

Do not ignore dangling nodes. Sinks must redistribute their rank or be treated as linking to all pages.

中文：不要忽略 dangling nodes。Sink 的分数必须平均分配，或者把 sink 当作连向所有页面。
